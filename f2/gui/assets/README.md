# GUI v02 资源文件

此目录用于存放GUI资源文件,如:
- 图标文件 (.svg, .png)
- 图片资源
- 字体文件

## 建议的资源结构

```
assets/
├── icons/
│   ├── douyin.svg
│   ├── tiktok.svg
│   ├── weibo.svg
│   ├── twitter.svg
│   └── ...
├── images/
│   └── ...
└── fonts/
    └── ...
```

## 图标来源

可以从以下网站获取Material Design图标:
- https://fonts.google.com/icons
- https://materialdesignicons.com/
